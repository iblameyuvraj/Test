## ✨ Code Editor Features
This is a powerful, cloud-synced IDE designed for a seamless development experience:

- ☁️ Cloud Persistence: Integrated with Supabase Storage. Your files and folders are saved in the cloud and persist across sessions.
- 💾 Auto-Save & Manual Save: Never lose work! The editor auto-saves every 5 seconds, and you can also use the manual SAVE button.
- 🚀 Real-time Execution: Run your code (Python, JavaScript, etc.) directly in the browser via high-speed WebSockets.
- 📂 Workspace Management: Full support for creating, renaming, and deleting files and nested folders.
- ⚠️ Data Loss Prevention: Smart "Dirty State" tracking warns you if you try to close or switch files with unsaved changes.
- 🌓 Dynamic Themes: Toggle between Dark and Light modes for your preferred coding environment.
